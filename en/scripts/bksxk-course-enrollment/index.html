Course Selection Helper

西农抢课助手 Pro

Tracks target teaching-class IDs on the NWAFU course-selection site, retries enrollment requests, and notifies results.

Full English notes are not available yet. The Chinese README is shown below.
This script is marked as paused. Read the notes before use.

用于西北农林科技大学本科生选课系统。维护目标教学班列表，按设定间隔轮询并重试提交，处理结果通过声音或桌面通知提醒。

当前归档说明：该脚本近期测试存在较大问题，暂不建议继续安装或依赖自动抢课功能。目录和安装地址保留，便于已安装用户查看历史说明；如后续完成修复，可重新验证后恢复维护状态。

功能

  • 维护一组目标教学班 ID，并按设定间隔尝试处理。
  • 支持课程类型、请求间隔、重试频率、成功后停止、教材订购策略等配置。
  • 在页面上显示运行面板，便于添加目标、查看日志和调整参数。
  • 设置面板提供稳妥、常规、快速三档速度预设，也保留高级请求参数。
  • 登录页可保存本机账号密码；运行中掉线后会等待浏览器自动填充，再只补空字段，手动输入验证码满 4 位且稳定后按原样提交，并在能确认可选轮次时恢复进入选课。
  • 可用声音和桌面通知提示处理结果。

适用范围

  • 网站：西北农林科技大学本科生选课系统。
  • 页面：https://bksxk.nwafu.edu.cn/*。
  • 浏览器：支持 Tampermonkey 的现代浏览器。
  • 脚本管理器：Tampermonkey（已测试）。

使用方法

安装后进入选课系统，在页面面板中添加目标教学班并启动。建议先用较长的请求间隔确认配置无误，再按实际情况调整。

已知限制

  • 需要你知道目标教学班 ID 和课程类型。也支持选择课程直接添加进列表。
  • 遇到需要确认教材的教学班时，脚本会按设置中的教材策略提交。默认策略是不订购，并使用“已购买正版教材”作为原因。
  • OCR 验证码识别默认关闭，只作为可选辅助；启用后最多尝试 3 张验证码，失败后按提醒设置提示手动输入。
  • 如果登录后没有上次轮次记录，且页面没有明确标出当前可选轮次，脚本会停下提示手动选择，不会自动选第一个轮次。
  • 页面接口、登录流程或验证码样式变化后，部分功能可能需要跟进。
